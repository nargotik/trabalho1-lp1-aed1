116 graph TB

165 ### Função is_primo:
166 Foi implementada a função is_primo tentando usar uma forma mista do ***Crivo de Eratóstenes*** com a forma mais vulgar de mediante um ciclo até à raiz quadrada do numero a testar.
167 Utilizando o ***Crivo de Eratóstenes*** logo numa primeira instancia aceitamos os primos que conhecemos
168 ```c
169  if (n==2)
170  return true;
171 ```
172 E descartamos todos os múltiplos dele próprio
173 ```c
174  // Todos os pares não sao primos if (n%2 == 0)
175  return false;
176 ```
177 É efetuada a mesma operação para todos os seguintes até ao numero 19:
178 ```c
179  // Proximos numeros primos
180  if (n==3 || n==5 || n==7 || n==11 || n==13 || n== 17 || n==19)
181  return true;
182  // Verifica o resto da divisao por um deles possivel
183  if (n%3==0 || n%5==0 || n%7==0 || n%11==0 || n%13==0 || n%17==0 || n%19==0 )
184  return false;
185 ```
186 Após esta implementação apenas será necessário testar um numero reduzido de hipóteses.
187 ```c
188  // Caso nao seja um caso destes itera
189  // Para i=19 ate i<=sqrt(n)
190  int tocheck = sqrt(n);//n / 2; // se n>n/2 entao n % n > 0
191  for (int i = 19; i < tocheck; i = i + 2) { // Os pares ja foram todos verificados
192  // Verifica se é divisivel por i
193  if (n % i ==0)
194  return false;
195  }
196 ```
197 Conclusão: Sobre a função is_prime conseguiu-se obter os números primos até 1,000,000 em cerca de ***1 Segundo*** poupando assim iterações desnecessárias
198 ### Função gera_sequencia:
199 Esta função tem por objectivo gerar um array de tamanho variável definido num parâmetro cujos números estão cumpridos entre o parâmetro min e max.
200 ```c
201 /**
202  * Função que gera uma quantidade de numeros aleatorios definida nos parametros
203  * @param min numero minimo a gerar
204  * @param max numero maximo a gerar
205  * @param numeros quantidade de numeros
206  * @return pointer do array de numeros
207  */
208 int * gera_sequencia(int min, int max, int numeros) {
209  int *numero_aleatorio;
210 
211  numero_aleatorio = (int*) calloc(numeros, sizeof(int));
212 
213  srand((unsigned) time(NULL));
214  for (int i=0; i < numeros ;i++)
215  numero_aleatorio[i] = rand() % max + min;
216  return numero_aleatorio;
217 }
218 ```
219 É utilizado a função rand() que necessita de um seed para ser pseudo-random que é chamado pela função srand().
220 
221 A função faz uso de de apontadores de endereço de memoria de forma a que seja passada no return o apontador do endereço.
222 
223 Para evocar a função pode ser da seguinte forma:
224 ```c
225  int *numeros_aleatorios;
226  // Chama função que gera a quantidade necessaria.
227  numeros_aleatorios = gera_sequencia(NUMERO_MIN, NUMERO_MAX, NUMEROS);
228 ```

236 /**
237  * Função somatorio questao 08 usando uma subrotina
238  * @param numero Numero processar
239  * @return somatorio dos pares
240  */
241 unsigned int somatorio(int numero) {
242  unsigned int sum=0;
243  for (int i=1 ;i<=numero ;i++) {
244  if (par(i))
245  sum += i;
246  }
247  return sum;
248 }
